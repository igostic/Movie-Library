### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/OmkarK05/BookMyShow.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. To start project
   ```sh
   npm start
   ```
4. To run unit tests
   ```sh
   npm test
   ```
